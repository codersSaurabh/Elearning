What is Identifiers ?

Identifiers are the names given to diffferent parts of the program viz. variables, objects, classes, function, lists, dictionaries and so forth

Naming rules for Python Indefiers

  1. Variable name must only be a non-keyword word with no spaces in between.
  2. VAriable names must be made up of letters,numbers and underscore(_).
  3. Variable name cannot befin with a number, although they can contain numbers.
  4. No specific character is allow to use except( _ ).
  5. Identifier can begin only with letters or underscore( _ ) anything else is not allowed at beginning.
