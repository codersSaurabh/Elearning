or

Creating Variable

  • python has no command for declaring the variable.
  • In python you don't need to declare the data type for a variable
  • When you make a variable in python you first assign value to it.

Some example of valid variable assigning.

  • student="raghav"
  • Student="ramesh"

    • Note: Python is a case sensitive language so in above example (student) and (Student) both are different variables.

  • a=b=c=6

    • In above example we assigning value 6 to variable a,b and c.

  • x,y,z=10,20,30

Invalid variable assigning

  • a=9,b=20,c=89
  • int a=56
