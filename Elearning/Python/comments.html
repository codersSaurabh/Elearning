Comments

Comments are used to understand the code.

Comments use to make program more readable.

Comments are ignore by the interpreter.

Comments help to understand the code written by another programmer.

Creating a comment

  • # use to make an single line comment.

Example

#printing an statement
print("hello world")
a=90 #assigning value to a

Multiline comments

Generally python does not any syntax for making multiline comment
For adding multiline comment you could insert # for each line.

#python comment
#another comment
#more than two comments
print("hello world")

Or we can use multiline string to add a multiline comment
Since python ignore string literal that are not assigned to a variable, you can add a multiline string (triple quotes) in your code and place your comment inside it.

"""
this is a
multine
comment
"""
print("hello World!")
