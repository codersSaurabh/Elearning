Data type

Data type is the type of data that is hold by a variable

To check the data type of any variable use the operator type()

1. Data types for Number

  • (a)Integer: It is the normal representation of wholle nummbers. The integer can be positive or negative

    (b)Boolean: These represent the truth values False and True. The Boolean type is a subtype of plain integers, and Boolean values False and True behave like the values 0 and 1, respectvely.

  • Floating point Numbers: In Python, floating point numbers represent machine-level double precision floating point numbers.
  • Complex number: Python represents complex numbers in the form A+B j. Complex numbers are a composite quantity made of two parts: the real part and the imaginary part, both of which are represented internally as float values.

2. Data type for strings

  • All strings in Python are sequences of pure unicode cahractes.
  • an string can hold any type of known characters

    • Example of legal strings in Python -

    "abcd", "1234", '@@@', '$%&^*'

    Each character of string can be individually accessed by index.

3. Lists

  • A List in Python represents a group of comma-separated values of any data types between square brackets.

    • Example-

    [1,2,3,4]
    ['a','i','o','p']
    ["python",8.21,8]

4. Tuples

  • Tupels are represented as group of comma-separated values of any data types within paranthesis. We will discussed about the tuples later in this tutotrial

    • Example -

    p=(1,2,5,6)
    tpl=('t','g','ram')
    mix=(78,1,"hello")

5. Dictionaries

  • The dictionary is an unordered set of comma-separated key:value pairs, within { }
  • In dictionary two same keys are not allowed.

    Example -

    {"India":1, "China":2, "America":3}
