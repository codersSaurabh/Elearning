Type Casting in Python

Type casting in Python refers to the process of converting one data type into another. Python provides several built-in functions for type casting, allowing you to change the data type of a variable as needed.

Here are some common type casting functions in Python:

Data type Function Example
int(): Converts a value to an integer int(5.6) will result in 5.
float(): Converts a value to a floating-point number float("3.14") will result in 3.14.
str(): Converts a value to a string str(42) will result in "42".
list(): Converts an iterable (e.g., a tuple or string) to a list list((1, 2, 3)) will create a list containing [1, 2, 3].
tuple(): Converts a list or an iterable to a tuple tuple([1, 2, 3]) will create a tuple (1, 2, 3).
bool(): Converts a value to a boolean. It's common to use bool() for truthiness checks. bool(0) results in False, and bool(42) results in True.


Note: Type casting is a powerful feature that allows you to manipulate and work with different data types as needed
in your Python programs. It's important to use these functions carefully to avoid potential errors and unexpected behavior.
