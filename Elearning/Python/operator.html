Operators in Python

Operators in Python are special symbols or keywords that are used to perform operations on operands. Operands can be variables, constants, or expressions. Python supports a wide range of operators to perform different types of operations.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations:

  • + - Addition (e.g., 5 + 3 = 8)
  • - - Subtraction (e.g., 7 - 2 = 5)
  • * - Multiplication (e.g., 4 * 6 = 24)
  • / - Division (e.g., 10 / 2 = 5)
  • % - Modulus (e.g., 9 % 4 = 1)
  • ** - Exponentiation (e.g., 2 ** 3 = 8)
  • // - Floor Division (e.g., 7 // 3 = 2)

Comparison Operators

Comparison operators are used to compare values:

  • == - Equal to (e.g., 5 == 5 is True)
  • != - Not equal to (e.g., 3 != 5 is True)
  • < - Less than (e.g., 2 < 3 is True)
  • > - Greater than (e.g., 4 > 2 is True)
  • <= - Less than or equal to (e.g., 5 <= 5 is True)
  • >= - Greater than or equal to (e.g., 4 >= 5 is False)

Logical Operators

Logical operators are used to combine logical values:

  • and - Logical AND (e.g., True and False is False)
  • or - Logical OR (e.g., True or False is True)
  • not - Logical NOT (e.g., not True is False)

Assignment Operators

Assignment operators are used to assign values to variables:

  • = - Assign (e.g., x = 5 assigns 5 to variable x)
  • += - Add and assign (e.g., x += 3 is equivalent to x = x + 3)
  • -= - Subtract and assign (e.g., x -= 2 is equivalent to x = x - 2)
  • *= - Multiply and assign (e.g., x *= 4 is equivalent to x = x * 4)
  • /= - Divide and assign (e.g., x /= 2 is equivalent to x = x / 2)
  • %= - Modulus and assign (e.g., x %= 3 is equivalent to x = x % 3)

Bitwise Operators

Bitwise operators are used to perform operations on binary values:

  • & - Bitwise AND (e.g., 5 & 3 = 1)
  • | - Bitwise OR (e.g., 5 | 3 = 7)
  • ^ - Bitwise XOR (e.g., 5 ^ 3 = 6)
  • ~ - Bitwise NOT (e.g., ~5 = -6)
  • << - Left shift (e.g., 5 << 2 = 20)
  • >> - Right shift (e.g., 5 >> 2 = 1)

Membership Operators

Membership operators are used to test if a value is present in a sequence:

  • in - Present in the sequence (e.g., 3 in [1, 2, 3] is True)
  • not in - Not present in the sequence (e.g., 4 not in [1, 2, 3] is True)

Identity Operators

Identity operators are used to compare the memory location of two objects:

  • is - True if the operands are identical (e.g., x is y is True if x and y reference the same object)
  • is not - True if the operands are not identical (e.g., x is not y is True if x and y do not reference the same object)
